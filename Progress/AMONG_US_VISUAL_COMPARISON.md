```
╔═══════════════════════════════════════════════════════╗
║ ●                  ROOM NAME                      ●   ║  ← Top border + rivets
║                   Subtitle Text                       ║

║    ┌───┬───┬───┐    Grid Floor Pattern:              ║
║  ● │   │   │   │ ●  Vertical lines every 40px        ║
║    │ ● │ ● │ ● │    Horizontal lines every 40px      ║
║    │   │   │   │    Color: Base + 10% lighter        ║
║  ● │ ● │ ● │ ● │ ●  Alpha: 0.2                       ║
║    └───┴───┴───┘                                      ║
║                                                       ║
║ ●              🎨 (64px emoji, 30% alpha)         ●   ║
║                                                       ║
║ ●                                                 ●   ║
╚═══════════════════════════════════════════════════════╝
 ↑                                                   ↑
Rivets every 80px                         Inner border (metallic)
```

**Layers (Bottom to Top):**
1. **Base Fill:** Room color (e.g., `#6E5A7E`)
2. **Grid Pattern:** Lightened base color, 1px lines
3. **Inner Border:** `#8B9AA3` metallic panel (2px)
4. **Outer Border:** Room borderColor (4px, 0.8 alpha)
5. **Rivets:** `#4A5A62` circles (6px diameter)
6. **Labels:** White text with black stroke
7. **Icon:** Large emoji (64px, 30% alpha)
8. **Checkpoints:** Grid arranged (48×48 squares)
